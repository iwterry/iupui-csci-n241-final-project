Contact

Provides a form where you can submit comments about the website.

Mission

The mission of this website is to help get more people interested in machine learning and to be a resource for ways to learn more about (and do more with) machine learning.

Vision

The vision of this website is for it to be a light that can make the path towards learning and using machine learning easier. Regardless of their educational background, gender, income, etc., people should have more access to an education for machine learning. Machine learning must be more inclusive.

Goals

I have three major goals for this website:

  • Provide resources to explain what machine learning is and how it is being used in the world
  • Provide resources for practicing and knowing some fundamentals of machine learning
  • Provide some motivation for learning machine learning when the journey gets hard
